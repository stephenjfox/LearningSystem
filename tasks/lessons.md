# Lessons & Patterns

- **Thought Continuity:** Check `docs/thoughts/` at the start of every session to review long-term goals and architectural reflections.
- **Task Management:** Rigorously update `tasks/todo.md` and follow the "Plan First" mandate from `@WORKFLOW.md`.
- **Data Schema:** Always include migration logic when updating the `state` structure to prevent user data loss from `localStorage`.
